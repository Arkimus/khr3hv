区切りのある文字列を代入すると、リストとして登録します。 リストをカンマ区切りの文字列として読み出すことができます。

Examples
CommaTextを抜き出す
CopyC#
ByteList byteList = new ByteList ();
byte[] data = new byte[5] { 1, 2, 3, 4, 5 };

byteList.Bytes = data;
byteList.Delimiter = ":";
Console.WriteLine (byteList.CommaText); // 結果は 1:2:3:4:5
