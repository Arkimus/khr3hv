RCB-4 Command Interfase Class Library
Rcb4 Command Interface Class Libraryは近藤科学株式会社のRCB-4ロボットコントロールボードを 外部入力から制御するためのクラスライブラリです。
Namespaces
NamespaceDescription
Rcb4
Rcb4クラスには各種アドレス定義値や定数などが保存されています。また、CommandクラスにはRCB-4HVのファームウェアを外部駆動するための命令を作成するメソッドがあります。
